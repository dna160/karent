Local Setup Guide
Everything you need to run Karent on your machine
ℹ️ This HTML file is your local dev dashboard. No build step needed — just open it in a browser. The API must be running at http://localhost:3001 for it to work.
1 · Start Postgres + Redis (Docker)
cd "D:\Claude Home\Karent"
docker-compose up -d
2 · Install API deps + push DB schema
cd apps/api
npm install
npx playwright install chromium   # download headless/headed Chromium
node node_modules/prisma/build/index.js db push
db push creates the influencers table in the public schema. Per-account schemas are created automatically when you add an account via the API.
3 · Configure your .env
# Root .env (already has defaults):
DATABASE_URL=postgresql://karent:karent@localhost:5432/karent
REDIS_URL=redis://localhost:6379
PLAYWRIGHT_HEADLESS=false   # ← shows browser window locally
4 · Start the API
cd apps/api
node index.js
# or for auto-restart on changes:
npx nodemon index.js
API will be at http://localhost:3001/health
5 · Add the Melody account
# Option A — via this dashboard:
# Click "+ Add account" in the sidebar.

# Option B — via curl:
curl -X POST http://localhost:3001/api/accounts \
  -H "Content-Type: application/json" \
  -d '{"slug":"melody","display_name":"Melody","instagram_handle":"@melody"}'

# Then fill in credentials:
# Edit: config/accounts/melody/.env
